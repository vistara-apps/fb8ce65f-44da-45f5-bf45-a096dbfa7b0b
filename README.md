# Legalo - AI-Powered Legal Assistant

![Legalo Logo](https://static.tildacdn.com/tild3061-6339-4131-a531-663735363664/new_logo.png)

## Overview

Legalo is an intelligent legal companion designed to empower individuals with easy-to-understand legal advice and support. Built in collaboration with UK lawyers and advisors, Legalo democratizes legal knowledge and provides a user-friendly platform for all your legal needs.

## Features

### 🤖 AI Legal Assistance
Our AI-powered platform provides instant legal assistance, making complex legal matters easy to understand and manage.

### 💬 Chat with Intelligent Bot
Engage in natural language conversations with our intelligent chatbot for quick and accurate answers to your legal questions.

### 🛡️ UK Legal Compliance
Built with UK legal standards and created in collaboration with UK lawyers and advisors.

### 🧠 Machine Learning Powered
Embracing the latest innovations in machine learning to provide comprehensive coverage of most legal issues.

### ⏰ 24/7 Availability
Get legal assistance whenever you need it. Our AI assistant is available around the clock.

### 👥 Accessible to Everyone
We believe legal assistance should be accessible to everyone. Our platform democratizes legal knowledge for all.

## Technology Stack

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS with custom design system
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **UI Components**: Headless UI

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/your-org/legalo.git
cd legalo
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Design Principles

### 80/20 Approach
Following the 80/20 principle, we focus on the most impactful UI/UX improvements:

1. **Visual Hierarchy** (20% effort, 80% impact)
   - Clear typography scale
   - Consistent spacing system
   - Strategic use of color and contrast

2. **User Flow Optimization** (20% effort, 80% impact)
   - Simplified navigation
   - Clear call-to-action placement
   - Intuitive chat interface

3. **Performance & Accessibility** (20% effort, 80% impact)
   - Fast loading times
   - Keyboard navigation support
   - Screen reader compatibility
